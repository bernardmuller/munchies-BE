MUNCHIES API

Routes

Authentication routes
  • post /register - requires email and password
  • post /login - requires email and password, sets LOGGED_IN_USER cookie => returns JWT token and user ID
  • get /auth - decode JWT => returns user object with auth = true or false
  • post /logout - clears all cookies
meals routes
  • get /meals - requires JWT => returns list of all meals
  • get /meals/:mealID - requires JWT => returns meal object
  • put /meals/:mealID/edit - requires JWT => updated meal with req.body
  • post /meals/create - requires JWT => creates new meal from req.body
  • delete /meals/:mealID - requires JWT => deletes meal
